# `mpcli`

## Thin command line interface to `mpd`.

### Usage:
`mpcli [ COMMAND ] [ ARGS ]`

If no command is given, the command `commands` is used.

Commands are passed directly to mpd through python-mpd.

The results are pretty-printed using Python's pprint module.
